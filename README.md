# Social REST API
## Python 3.8, Django 3

This is a simple REST Api for a news aggregator.

```
pip install -r requirements.txt
```
## ENV config
ENV config can be set up by:
```
cp .env-example .env
```
## Run docker
```
docker-compose build
docker-compose up
```
